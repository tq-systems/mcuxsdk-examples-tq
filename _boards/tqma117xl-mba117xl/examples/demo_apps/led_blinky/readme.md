# Led Blinky Demo

## Overview

This LED Blinky demo application provides a sanity check for the new SDK build environments and board bring-up. 
The demo uses SysTick interrupts to implement timing delays. In this example, the user LED 1 on the MBa117xL is toggled periodically.
The purpose of this demo is to provide a simple project for debugging and further development.


## Demo Output

User LED 1 on MBa117xL is toggled periodically.
